Workshop: Interoperable Workflows Using Workflow Management Systems and Singularity Containers

Wednesday September 6, 2023 -- 11 am – 5 pm Eastern time
Application deadline is August 25, 2023.

Sponsored by the National Institute of Health’s Common Fund Data Ecosystem, and in partnership with the HuBMAP Consortium and the Brain Image Library, PSC is proud to present a workshop on using Common Workflow Language (CWL) to create interoperable workflows.

More information: https://www.psc.edu/events/workshop-interoperable-workflows-using-cwl/

Past Workshops and Training Events


Data Submission Virtual Workshop

June 27, 2023

This virtual workshop presents a hands-on walk through of the data submission process and how to use the BIL Analysis Ecosystem for pre-submission exploration, sharing, and processing. The targeted audience for the workshop includes data submitters, PI's, data managers, and microscopy lab personnel. Topics covered include:

  • BIL data scope and available resources for data submitters
  • Hands-on introduction to the data submission process, including:
    • Gaining data submission access
    • Using the submission portal for metadata
    • Transferring data to BIL
  • Validating and releasing data to the public
  • Using the BIL Analysis Ecosystem for pre-submission exploration and processing

Day 2: Data Exploration Virtual Workshop
April 4, 2023
Recording: https://www.youtube.com/watch?v=0Z-rkQEmrkY

This virtual workshop presents a hands-on introduction to the data in BIL and the BIL Analysis Ecosystem. The target audience are individuals interested in accessing or using the microscopy data contributed to BIL. Topics covered include:

  • Introduction to the data in BIL and how to find it
  • How to access the various BIL resources for in-place data exploration
  • Using the BIL Analysis Ecosystem to run Jupyter notebooks, visualize and analyze data using a variety of software tools
