Training and Workshops

Data Submission Workshop

üìÜ When: January 29, 2026
üïê Time: 1-3PM ET
üìç Where: Virtual (Zoom)

This virtual workshop presents a hands-on walk-through of the data submission process and how to use the BIL Analysis Ecosystem for pre-submission exploration, sharing, and processing. The targeted audience for the workshop includes data submitters, PI's, data managers, and microscopy lab personnel. Topics covered include:

  • Overview of BIL data scope and available resources for data submitters
  • Hands-on introduction to the BIL data submission process
  • Steps to obtain data submission access
  • Using the submission portal to enter and manage metadata
  • Transferring and uploading data to BIL
