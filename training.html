Data Submission Virtual Workshop
Next Workshop: July 27th-28th
Signup Link: https://forms.gle/nTqXRLpzhR7Z4Kw37

This workshop, presents a hands-on walk through of the data submission process in use by the Brain Image Library along with how to explore and visualize Brain Image Library data with the tools available. Topics covered include:

  • BIL scope and available resources for data submitters
  • Hands-on introduction to the data submission process, including:
    • Gaining data submission access
    • Using the submission portal for metadata
    • Transferring data to BIL
  • Validating and releasing data to the public
  • Using BIL computational and visualization resources for pre-submission exploration and processing
The targeted audience for the workshop includes data submitters, PI's, data managers, and microscopy lab personnel. Day 1 will consist of Data Submission walkthrough and Day 2 will consist of a Data Exploration and Visualization exercise.
